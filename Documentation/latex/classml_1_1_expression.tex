\doxysubsection{Detailed Description}
表达式的基类。表达式用于计算。每个表达式\+: 


\begin{DoxyItemize}
\item 最基本的求值当然得有
\item 有一个名字，代表这个量在我们软件中的意义
\item 有一个描述，是对某个量更加详尽的描述
\item 可以转换为字符串，以便于显示
\end{DoxyItemize}

例如： J \texorpdfstring{$\ast$}{*} (pr -\/ pwf)

这个表达式的名字是\"{}产液量\"{} 描述：\"{}产液量，不考虑泡点压力\"{}
