◆ setDate1904()

void Workbook::setDate1904 ( bool  date1904)

Excel for Windows uses a default epoch of 1900 and Excel for Mac uses an epoch of 1904. However, Excel on either platform will convert automatically between one system and the other. Qt Xlsx stores dates in the 1900 format by default.

Note
This function should be called before any date/time has been written.
