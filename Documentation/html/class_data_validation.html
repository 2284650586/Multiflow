◆ ValidationType

enum DataValidation::ValidationType

The enum type defines the type of data that you wish to validate.

\value None the type of data is unrestricted. This is the same as not applying a data validation. \value Whole restricts the cell to integer values. Means "Whole number"? \value Decimal restricts the cell to decimal values. \value List restricts the cell to a set of user specified values. \value Date restricts the cell to date values. \value Time restricts the cell to time values. \value TextLength restricts the cell data based on an integer string length. \value Custom restricts the cell based on an external Excel formula that returns a true/false value.
